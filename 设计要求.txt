大学生个人成长与学业档案数字化系统 设计需求文档
一、基本要求
开发完整的大学生个人成长与学业档案数字化系统，实现学生在校期间个人资料、学业成绩、科研成果、获奖记录、实习经历等数据的集中化管理，引入企业角色，完成学校与用人单位的信息交互。
系统核心目标
为学生提供全面记录与展示个人成长与能力的专属平台；
为导师提供学生档案审核与成长指导专属通道；
为企业 HR 提供实习招聘、学生能力评估与反馈接口，实现校企高效互动；
为学院、学校提供可视化统计、成长评估及毕业综合分析报告。
附加要求
系统界面设计美观、操作简洁、功能清晰、交互友好，兼顾实用性与扩展性；开发过程需查阅相关文献，参考国家数据安全与个人隐私保护标准，按时完成各阶段工作任务，严格按照毕业论文格式要求撰写论文与说明文档。
二、设计参数
开发工具：IntelliJ IDEA + Visual Studio Code + MySQL
开发框架：Spring Boot + Vue3 + MyBatis-Plus
系统架构：前后端分离架构，提供RESTful API接口
用户界面：包含学生端、导师端、企业 HR 端、管理员端四类独立界面
核心功能：实现档案数据导入导出（Excel/PDF）、报表生成与可视化展示（ECharts）
页面要求：布局合理、界面美观、操作流畅
三、设计内容及工作量
（1）用户与权限管理模块
实现学生、导师、企业 HR、管理员多角色登录与权限分配功能；支持用户注册、登录、修改密码、角色切换及细粒度访问权限控制。
（2）学生档案管理模块
核心管理学生个人信息、学业成绩、课程表现等数据；支持学业成绩导入导出、自动绩点计算、成绩趋势可视化展示；学生可上传个人简历、获奖证书、科研项目、社会实践材料、成长日记。
（3）成长记录与作品展示模块
为学生提供自我展示空间，支持作品集、学习成果、心得体会上传；导师可对成长记录添加批注与指导意见，系统自动归档留存；支持生成成长档案 PDF 报告，方便学院、企业查阅使用。
（4）企业对接模块
企业 HR 完成注册认证后，获取对应权限，可浏览学生档案、开展学生能力评估等操作，实现校企数据互通。
（5）数据统计与成长评估模块
系统对学业成绩、获奖情况、科研成果、实习表现等学生成长指标进行加权分析，自动生成综合成长分；提供雷达图、柱状图、趋势图等多维度可视化图表，辅助学院掌握学生整体成长态势。
（6）报表导出与数据分析模块
支持生成学生个人成长档案报告（PDF/Word 格式）与学院综合统计报表；提供标准化数据导入导出功能，支持 Excel 模板下载、批量数据上传。
（7）可扩展功能
支持企业发布招聘信息，实现企业与学生在线交流互动。
